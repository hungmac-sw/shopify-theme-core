# shopify-theme-core

**Core Structures**
```
 |-- resources
    |-- app
        |-- app.js
        |-- app.css
    |-- config
    |-- module
        |-- templates
            |-- page
                |-- page.liquid    
                |-- page.scss
                |-- page.js
            ...
    |-- scss
        |-- main.scss
        |-- variables.scss
 |-- src
```

Learn how to configuring theme settings
```https://shopify.dev/tutorials/develop-theme-theme-editor-settings-schema#input-setting-types```

Learn about Liquid
```https://shopify.github.io/liquid/```

For build js and css
```npm run dev```

Deploy files has changed
```theme watch```

Deploy all files
```theme deploy```
